cl-booth-library-manager

Common Lisp と TypeScript (WebView2) を組み合わせて開発された、Windows 向けの BOOTH ライブラリ管理アプリケーションです。

概要

本プロジェクトは、Common Lisp の学習の一環として作成された個人用アプリケーションです。BOOTH で購入したアイテムの管理を、デスクトップ上でシームレスに行うことを目的としています。

ダウンロード

最新の実行ファイル（EXE形式）は、以下の Releases ページよりダウンロードいただけます。

Download cl-booth-library-manager (Releases)

技術スタック

  • Backend: Common Lisp (ASDF, CFFI)
  • Frontend: TypeScript / HTML / CSS (WebView2 上で動作)
  • OS: Windows

主な機能と仕組み

1. WebView による認証とリスト化

アプリケーション起動後、「ログイン」ボタンを押下することで WebView2 ウィンドウが立ち上がります。

  • ユーザーが自身の BOOTH アカウント情報を入力してログインすることで、ライブラリへのアクセスに必要な Cookie を取得します。
  • 取得した Cookie をもとに、ライブラリ一覧を自動的に取得・表示します。

2. セキュリティとプライバシー

ユーザーの認証情報の保護の為、以下の設計を行っています。

  • 暗号化保存: 入手した Cookie はプレーンテキストではなく、ローカルディレクトリ内に暗号化された状態で保存されます。
  • 外部送信の禁止: 取得した認証情報やライブラリのリストが、開発者のサーバーを含む外部の第三者へ送信されることは一切ありません。通信は BOOTH (pixiv) 公式ドメインとの間でのみ行われます。

3. 手動登録モード

「WebView 経由でアカウント情報を入力することに抵抗がある」という場合や、特定のアイテムのみを管理したい方向けに、手動登録機能を備えています。アカウント連携を行わずに、ローカルの管理ツールとして利用することも可能です。

開発にあたって

このリポジトリは、Lisp による Windows デスクトップアプリ開発のプラクティス（CFFI を用いた Win32 API/WebView2 の呼び出しや、TypeScript 製フロントエンドとの連携）を模索するために作成されました。
